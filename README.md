# FinSight — ML-Powered Personal Finance Analyzer

FinSight is a full-stack web application that uses machine learning to detect 
anomalous transactions in personal finance data. It combines an Isolation Forest 
model with a FastAPI backend and a React dashboard to give users real-time 
spending insights and fraud alerts.

## Stack
- **ML:** scikit-learn (Isolation Forest), Pandas, NumPy
- **Backend:** Python, FastAPI, PostgreSQL, SQLAlchemy, JWT Auth
- **Frontend:** React, Recharts, Axios
- **DevOps:** Docker, Docker Compose, GitHub Actions, Render, Vercel
